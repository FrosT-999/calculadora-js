# Calculadora JS (Factory & Constructor Functions)

Projeto de uma calculadora funcional desenvolvida durante o curso de JavaScript e TypeScript (Udemy). O foco do projeto foi praticar lógica de programação, manipulação do DOM e diferentes padrões de criação de objetos.

## 🚀 Tecnologias Utilizadas
* JavaScript (ES6+)
* HTML5
* CSS3

## 🧠 Conceitos Praticados
* **Factory Functions**: Criando objetos que retornam a lógica da calculadora.
* **Constructor Functions**: Refatoração do código para usar o operador `new` e `this`.
* **Escopo e `this`**: Uso de Arrow Functions para travar o escopo léxico.
* **Tratamento de Erros**: Implementação de `try/catch` para cálculos inválidos.
* **Eventos**: Captura de cliques e teclas (Enter) de forma global no `document`.

## 🛠️ Como rodar o projeto
1. Clone o repositório:
   git clone https://github.com/seu-usuario/calculadora-js.git
2. Abra o arquivo `index.html` no seu navegador.
